# Analise-de-Dados-de-E-commerce-com-Databricks
Este projeto tem como objetivo analisar dados reais de um e-commerce brasileiro, construindo um pipeline completo de dados e gerando insights estratégicos para o negócio.  A solução foi desenvolvida utilizando Databricks, com foco em organização de dados em camadas (Bronze, Silver e Gold) e análises voltadas para tomada de decisão.
